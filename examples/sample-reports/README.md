# Sample Reports

這裡只放人工挑選的正式樣本，用來做三件事：

- 展示目前 skill 的輸出品質
- 當作報告契約的回歸基準
- 讓 PR reviewers 不用去翻外部執行目錄

## Rules

- 官方執行輸出不放這裡。
- 官方執行輸出固定在：
  - `%USERPROFILE%\tw-sector-screener-output`
- 這裡只保留少量樣本，避免 repo 被大量報告灌爆。
- 若 PR 影響報告、audit、validation、watchlist 契約，請同步更新對應樣本。
